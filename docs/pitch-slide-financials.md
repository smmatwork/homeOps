### Revenue Model — 5 Streams, Layered by Phase

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  STREAM 1: SaaS Subscription (Phase 1)                               │
│  ├─ Household plan: ₹299-499/month                                  │
│  ├─ Covers: AI agent, chore management, helper tracking, calendar   │
│  ├─ Margin: 90%+                                                     │
│  └─ Target: 10,000 HH × ₹400 = ₹4.8 Cr ARR (Year 1)             │
│                                                                      │
│  STREAM 2: Helper Premium (Phase 2)                                  │
│  ├─ ₹99-199/helper/month (paid by household)                       │
│  ├─ Covers: multi-HH coordination, check-ins, compensation ledger  │
│  ├─ Margin: 85%                                                      │
│  └─ Avg 3 helpers/HH × ₹149 = ₹447/HH/mo additional              │
│                                                                      │
│  STREAM 3: Maintenance Vendor Leads (Phase 3)                        │
│  ├─ ₹200-500 per qualified booking                                  │
│  ├─ Covers: AC service, plumbing, pest control, deep clean          │
│  ├─ Margin: 70%                                                      │
│  ├─ Avg 2 bookings/HH/quarter = ₹233/HH/mo                        │
│  └─ At 20K HH: ₹5.6 Cr ARR                                        │
│                                                                      │
│  STREAM 4: Community Marketplace Commission (Phase 4)                │
│  ├─ 5-8% of transaction (vs. UrbanClap's 30-40%)                   │
│  ├─ Covers: aggregated service bids, bulk vendor contracts          │
│  ├─ Margin: 60%                                                      │
│  ├─ Avg ₹5,000 GMV/HH/month × 6.5% = ₹325/HH/mo                 │
│  └─ At 50K HH: ₹19.5 Cr ARR                                       │
│                                                                      │
│  STREAM 5: Procurement Aggregation (Phase 4)                         │
│  ├─ 2-3% of community bulk-order value                              │
│  ├─ Covers: groceries, cleaning supplies, consumables               │
│  ├─ Margin: 65%                                                      │
│  ├─ Avg ₹8,000 grocery+supplies/HH/mo × 2.5% = ₹200/HH/mo       │
│  └─ At 50K HH: ₹12 Cr ARR                                         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```
